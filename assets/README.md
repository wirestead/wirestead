# Assets

This directory contains project identity assets used by the core repository
README, repository profile, package metadata, and related project materials.

## Production Assets

- `wirestead-logo-horizontal.svg` / `.png`: primary logo for light backgrounds.
- `wirestead-logo-dark.svg` / `.png`: logo lockup for dark backgrounds.
- `wirestead-logo-monochrome.svg` / `.png`: single-color logo.
- `wirestead-mark.svg` / `wirestead-mark-512.png`: standalone mark.
- `wirestead-mark-monochrome.svg`: single-color standalone mark.
- `wirestead-favicon.svg`, `wirestead-favicon-64.png`,
  `wirestead-favicon-512.png`: favicon and app icon assets.

## Brand Sources

Editable source files and the brand board live in `brand/`. The production SVGs
in this directory are preferred for documentation and package metadata because
their wordmarks are path-based and font-independent.

External documentation, benchmark, and container images remain in their legacy
repository locations until those repositories are moved. Full documentation
images, generated Doxygen assets, tutorial screenshots, benchmark result charts,
and repo-specific workflow diagrams should remain in their owning repositories.
